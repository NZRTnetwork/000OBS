# 000OBS — Obsidian Business Vault Template

Structured Obsidian vault template for solo operators and small teams — schema, service codes, role codes, MOC pattern, and Claude Code integration.

See `obsidian-business-vault-template/` for the full template.

## What's inside
- Frontmatter schema with tags, role codes, service codes, status
- HOME.md (MOC) pattern for every folder
- Service code taxonomy (000API, 000DOL, 000NCL, etc.)
- Claude Code CLAUDE.md integration
- Example domain folder with guide and reference templates
